# 모델 구분법

* imgupload
    - 이미지를 업로드하고 확인할 수 있는 앱입니다.

* main
    - 메인홈페이지입니다. 핑구웹사이트 라는 텍스트가 출력됩니다.

* spacyapp
    - spacy모델을 처리하는 앱입니다.

* content
    - spacy모델이 들어있는 폴더입니다. 이환님의 1_15 모델이 적용되어있습니다.

* pingu
    - 이 웹 폴더의 전체적인 setting과 url에 접근할 수 있습니다.

* ocrtool
    - 정연님의 ocr모델을 구현하려는 앱입니다. 아직 미구현입니다.

* media
    - 불러온 이미지가 저장되는 저장소입니다.
